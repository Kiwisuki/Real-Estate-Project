# Real Estate Project

Real estate website for Vilnius flats with additional features:

- Price, Rent, ROI prediction
- Time series relationship plotter
- Feature relationship plotter
- Flat valuation

## Components:
- Scraper
- Data preprocessor
- ML model
- Cloud database
- APIs infrastructure
